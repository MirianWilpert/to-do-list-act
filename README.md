## ? Como Executar

1. Clone este repositório ou baixe os arquivos
2. Instale as dependências: `npm install`
3. Executar MongoDB em container:  `docker run --name mongodb -p 27017:27017 -d mongo:latest`
4. Verificar se está rodando:  `docker ps`
5. Build e execução: `npm run build` `npm start `
